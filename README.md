# 🎸 搖滾吉他樂理教室

圖形化的搖滾吉他樂理學習 App，五大單元：和弦圖鑑、音階教學、調性字典、歌曲進行、樂理練習。

- 43 組常用和弦：開放和弦、強力和弦（Power Chords）、封閉和弦（Barre Chords）、七和弦、掛留和弦
- SVG 指板圖：手指編號、空弦/悶音標記、封閉橫按、高把位起始格標示
- 點擊即發聲：Karplus-Strong 弦振動合成，支援刷弦與分解和弦兩種播放方式，無需載入音色檔
- 分類篩選 + 和弦詳細面板（組成音與音程級數、把位類型、練習提示）
- 和弦變化教室：以選中和弦的根音展示 13 種和弦怎麼變出來（C → Cm / C5 / Csus / Cadd9 / C6 / Cm6 / C7 / Cmaj7 / Cm7 / Cdim / Cm7♭5），附按法圖、標記被動過的音、可逐一試聽
- 調性字典：任一調（12 調 × 大調/自然小調）的 I~VII 級順階三和弦與七和弦，附組成音、功能角色說明、按法與試聽；支援斜線和弦（C/B、D/F#…）按法
- 音階教學：全指板（15 格）高亮 7 種音階（大小調、五聲、藍調、Dorian、Mixolydian），12 個根音、級數/音名切換、點音試聽、播放整條音階；可選把位（五聲 Box 1–5、七聲 7 個把位），把位外的音自動變暗
- 歌曲進行：18 組進行——日系五大（王道、小室、卡農、丸サ、4536251）＋借用和弦教學組（IVm 催淚彈、♭VI–♭VII 熱血終止、副屬和弦鏈、♯IVm7♭5、分數和弦低音下行）＋歐美經典，小節跟播高亮、附按法與 solo 音階建議
- 移調器：任一進行一鍵移到 12 個調，和弦名、按法圖、播放全部跟著換（先在 C 調練熟再移調）
- 節奏引擎：長音／8 Beat／16 Beat／琶音四種伴奏模式，合成鼓組（大鼓、小鼓 backbeat 固定 2/4 拍、hi-hat 切分）、BPM 60–200 可調；丸サ與 4536251 內建半小節換和弦的和聲節奏
- 色彩切換：任一進行一鍵在「原版／七和弦版／add9 版」間切換聽顏色差（V 級自動升級屬七）
- 転調模式：模擬「最後副歌升 Key」——每循環一輪整組升半音
- 樂理練習：組成音、音程辨認、和弦變化、認和弦、順階級數五種題型隨機出題（每輪 10 題），即時解說與試聽

完整規劃見 [PLAN.md](./PLAN.md)。

部署於 GitHub Pages，透過 `.github/workflows/deploy-pages.yml` 於 push 到 `main` 時自動建置部署。

## 開發

```bash
npm install
npm run dev       # 開發伺服器
npm run build     # 型別檢查 + 產出 dist/
npm run preview   # 預覽建置結果
```

技術棧：React 18 + TypeScript + Vite + Tailwind CSS v4 + Web Audio API。
